# 🦺 Hardhat & Safety Vest Detection using YOLO

This project builds a **computer vision model for construction safety monitoring** using the YOLO object detection framework.

The model is trained to detect safety equipment used by workers, including:

- Hardhat
- Safety Vest
- Person

The goal is to explore how **deep learning and computer vision** can be used to improve workplace safety by automatically detecting whether workers are wearing proper protective equipment.

---

# 📌 Project Overview

Construction sites are high-risk environments where proper safety equipment is essential. Manual monitoring can be inefficient and prone to human error.

This project demonstrates how **object detection models** can help automate safety monitoring by identifying workers and their safety gear directly from images.

---

# 📂 Dataset

The dataset contains annotated images of construction workers.

Classes included:

- Hardhat
- Vest
- Person

Annotations are provided in **YOLO format**.

---

# 🧠 Model

The model used in this project is:

**YOLO (You Only Look Once)** via the Ultralytics implementation.

Training configuration:

| Parameter | Value |
|-----------|------|
| Model | YOLO26 Small |
| Epochs | 20 |
| Batch Size | 16 |
| Image Size | 640 |
